Jeu vidéo

Description

Ce projet a été réalisé en 1ère année de BUT dans un cadre pédagogique.

L'objectif de ce projet était de créer un jeu vidéo, de sa conception à son implémentation.
Cette application devait avoir une interface utilisateur et chaque partie du code devait être testée.
Il fallait utiliser le FrameWork WPF ainsi que IUTGame (Réalisé par les enseignants).

Conditions de Travail

  • Groupe : 4 personnes (Martin De Lima, Julien Guyenet, Gibril Lakhdar, Alexandre Moreau)
  • Outils utilisés : Github, Visual Studio 2022, C#, WPF, Visual Paradigm
  • Temps donné : 11h de formation et 12h de projet tutoré

Résultats

Nous avons réalisé un jeu avec un joueur qui doit survivre le plus longtemps à des vagues d'ennemis, grâce à des armes sur la carte et des bonus qui apparaissent après la mort d'un ennemi.

Fonctionnalités réalisées :

  • Gestion du joueur (Récupération des armes, attaque, déplacement, mort)
  • Gestion des ennemis (Création, attaque, déplacement, mort)
  • Gestion des bonus pour le joueur
  • Génération de la carte de jeu
  • Tableau des scores
  • Menu options
