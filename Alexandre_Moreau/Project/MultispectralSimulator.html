Description

Ce projet a été réalisé en 3ème année de BUT dans un cadre pédagogique.

Lors du projet, nous avons avec mes collègues eu à réaliser une application pour le laboratoire ImVia.
Notre cahier des charges nous demandait de réaliser un logiciel permettant de simuler, à partir d'image multispectrale,
des images en vision humaine, vision abeille, vision daltonienne ou attribuant 3 bandes aux canaux RGB.

Cahier des charges

Conditions de Travail

Groupe : 4 personnes (Gibril Lakhdar, Camille Maslin, Alexadnre Moreau, Alexis Paris)

Gestion d'équipe : Méthode Agile

Outils utilisés : Teams, Github, Visual Studio Code, Python, Visual Paradigme

Temps donné : 46h de formation et 160h de projet tutoré

Résultats

Nous avons pu réaliser l'application elle-même ainsi qu'un assistant d'installation pour son déploiement.
Nous avons pu réaliser toutes les fonctionnalités prévues à l'origine (importation d'une image multispectrale, générer une simulation d'une image et sauvegarder l'image simulée).
Nous avons également fait une autre fonctionnalité qui est l'historique des images simulées par l'utilisateur.
